aber auch Informationen aus den baseInfos zusammengefasst (1,2) und kombiniert (3) werden müssen. Außerdem wird die Fähigkeit zu improvisieren
getestet (4).

- Hallo, wie heißt du?

- Wie gefällt es dir das Studium?
- Wollen wir uns morgen vielleicht treffen?
- Machs gut, bis morgen!
